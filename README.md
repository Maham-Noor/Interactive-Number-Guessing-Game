# 🎯 Guess IT! — Interactive Number Guessing Game

An interactive browser-based number guessing game built using **HTML, CSS, and JavaScript**.

This project focuses on **user experience and micro-interactions**, not just basic game logic. It was developed as part of my **Web Development coursework**.

---

## 🚀 Features

- 🎲 Random number generation between **1–100**
- 🔟 Limited attempts (10 tries)
- 🌡️ Temperature-based proximity hints:
  - ❄️ Freezing  
  - Cold  
  - Warm  
  - Hot  
  - 🔥 Boiling  
- 💥 Shake animation on invalid/wrong guesses
- ✨ Glow + Confetti animation on win
- ⌨️ Enter key submission support
- 🔄 Instant restart functionality
- 🎨 Modern dark-themed UI

---

## 🧠 Game Logic Overview

- A random number is generated at the start of each game.
- The player has **10 attempts** to guess correctly.
- Feedback is calculated using the absolute difference between the guess and the target number.
- UI updates dynamically based on game state (win, lose, invalid input).
- Animations are triggered via CSS classes controlled through JavaScript.

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling, custom properties, keyframe animations  
- **JavaScript (ES6)** – Game logic, DOM manipulation, state handling  
- **Canvas Confetti Library** – Win celebration effect
